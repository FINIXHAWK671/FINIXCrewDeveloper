Introduction

This section introduces ethical cyber security techniques, including the use of Google Dorks, ethical penetration testing, and cybersecurity tools for learning and development purposes.

Google Dorks

Google Dorks are advanced search queries that help find specific information online. Examples include:

  • filetype:pdf site:example.com → Finds PDFs on a website
  • intitle:"index of" passwords → Searches for directories with passwords
  • inurl:admin login → Finds admin login pages
